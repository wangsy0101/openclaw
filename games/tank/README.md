# 坦克大战 · Classic Mode（离线可玩）

打开 `index.html` 即可玩（纯静态，不需要服务器）。

## 操作

- 移动：方向键 / WASD
- 开火：J / Space
- 暂停：P
- 重新开始：R
- 手机：右侧方向键（按住移动）+ 中间“●”开火

## 经典机制（本版本实现）

- **基地(★)**：基地被子弹击中立即失败
- **敌军波次**：每关固定数量敌人，最多同时刷出 4 个，清场进入下一关
- **地形**：
  - 砖墙：可被打碎
  - 钢墙：需要高火力（升到 3 星）才能打碎
  - 水域：坦克不可通过，子弹也会被挡住
  - 草丛：遮挡（覆盖坦克/子弹，模拟经典“隐身草”效果）
  - 冰面：会打滑（松开方向仍会短暂滑行）
- **道具**（由携带加成的敌人掉落，拾取触发）：
  - ★ 升星（提升火力/射速/子弹速度）
  - ⛑ 护盾
  - 🕒 冻结敌军
  - 🧨 炸弹（清屏）
  - 🧱 铁锹（暂时把基地周围加固成钢墙）
  - ＋ 加 1 命
