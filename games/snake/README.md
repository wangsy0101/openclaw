# 贪吃蛇（离线可玩）

一个**纯静态**（不用服务器）的贪吃蛇小游戏：打开 `index.html` 就能玩。

## 玩法

- 方向键 / WASD：移动
- Space：暂停/继续
- R：重新开始
- 手机：可点右侧方向键或在画布上滑动

## 运行

直接双击打开：

- `snake/index.html`

或把仓库部署到 GitHub Pages（推荐），即可用链接打开。

## GitHub Pages 部署（可选）

1. GitHub 仓库 → **Settings** → **Pages**
2. Source 选 `Deploy from a branch`
3. Branch 选 `main`，Folder 选 `/ (root)` 或 `/(docs)`（看你放哪）
4. 保存后等待生成访问地址

> 如果你把 `index.html` 放到仓库根目录，Pages 会更省事。
